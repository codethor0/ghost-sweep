Ghost Sweep is now open source.

We built a community-maintained project to help job seekers report suspected ghost job postings with moderation, transparency, and privacy boundaries from day one.

What's live today (v0.1.1):

- Public site: https://codethor0.github.io/ghost-sweep/
- Community report form: https://forms.gle/PsjaYrbrCjAgZXjW8
- Open repository: https://github.com/codethor0/ghost-sweep

What v0.1.1 adds over the initial release:

- Fixes found during pre-launch E2E validation (live E2E seed discovery, readable form validation errors)
- Documented local validation summary for contributors running the Docker stack

What we're clear about:

- Reports are unverified allegations until moderation
- The full backend runs locally for contributors; it is not publicly hosted yet
- This is active development, not a finished production service

If you've lost time to postings that never lead anywhere, we'd rather you spend that time on roles with real hiring intent.

Contribute, report bugs, or follow the roadmap on GitHub. MIT licensed. Community maintained.
