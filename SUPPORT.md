# Support and community help

Ghost Sweep is a volunteer open-source project. Response times depend on maintainer availability.

## Where to get help

| Need | Channel |
| ---- | ------- |
| General questions and ideas | [GitHub Discussions](https://github.com/codethor0/ghost-sweep/discussions) |
| Bugs and scoped contributor work | [GitHub Issues](https://github.com/codethor0/ghost-sweep/issues) |
| Security vulnerabilities | [SECURITY.md](SECURITY.md) (do not open public issues) |
| Code of conduct concerns | [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) |
| Contribution workflow | [CONTRIBUTING.md](CONTRIBUTING.md) and [docs/contributor-onboarding.md](docs/contributor-onboarding.md) |

## Reporting a suspected ghost job

Use the public Google Form linked from https://codethor0.github.io/ghost-sweep/

Do **not** post private report details, contact information, or evidence in GitHub Issues or Discussions.

## What we cannot support today

- Paid contributor roles or employment guarantees
- Immediate review of every community report
- Public backend API access (local Docker only)
- Automated production import from Google Sheet
- Legal advice about employer disputes

See [docs/reporting-and-moderation-policy.md](docs/reporting-and-moderation-policy.md) for moderation boundaries.
